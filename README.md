# HeYYou Sails

Custom windsurfing sails, handmade on Bonaire by Youp Schmit (@youpschmit / @heyyousails).

Immersive one-page site: vanilla HTML/CSS/JS with Three.js (3D sail), GSAP + Lenis (Apple-style scroll), and real freestyle footage. No build step.

## Run locally
```bash
python3 -m http.server 8123
# open http://localhost:8123
```

## Deploy
Any static host (GitHub Pages, Vercel, Netlify). The repo root is the web root.
